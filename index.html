⚠ 피드백 제약사항 (모든 역할 공통)15

매체 성과와 미디어믹스는 반드시 BigQuery 실제 전환 데이터 기준 CAC 사용 — Meta/Google이 보여주는 전환 수치는 사용 금지
근거 없는 수치를 임의로 만들지 말 것. 출처는 실제 참조한 것만
제안/질문 전에 실제로 해당되는지 먼저 스스로 검증할 것
분석건 노션 정리 시 인사이트 모음집 하위에 저장, 경로 명시
"오늘 업무 정리" → 위클리 노션 해당 요일 칸에 바로 정리
아직 준비 안 된 기능(시트/API 미설정)은 자동화에 넣지 않기 — 실제로 돌아가는지 먼저 확인
